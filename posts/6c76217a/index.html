HashMap

  • HashMap，它存储了从K类型键到V类型值之间的映射关系。哈希映射在内部实现中使用了哈希函数 ，这同时决定了它在内存中存储键值对的方式。许多编程语言都支持这种类型的数据结构，只是使用了不同的名字，例如：哈希（hash）、映射（map）、对象（object）、哈希表（hash table）、字典（dictionary）或关联数组（associative array）等，这只是其中的一部分而已。

  • HashMap中的数据存储在堆上;

  • HashMap是同构的, 要求Key必须是同一种类型, Value是同一种类型;

常用方法

创建hashmap

通过collect方法创建hashmap

在元素类型为Tupl的Vector上使用Collect方法可以创建一个HashMap;

  • 要求Tupl有2个值:一个作为HashMap的Key, 一个作为Value;
  • collect方法可以把数据整合成多种集合类型,包含HashMap,所以使用collect创建HashMap需要显示的指定返回值类型为HashMap<_,_>
  1. 第一种
  1. 第二种使用2个vec通过zip方式合并后通过collect方法创建;

insert(key, value) 插入数据

insert方法,在Map中插入键值对。如果Map中不存在key，则返回None。如果Map中存在此key，则会更新该值，并返回旧值(Some<Value>)。

Option get(key) 获取值, 可以通过match表达式取值

更新值

  • insert(key, value) 插入数据(存在数据覆盖, 不存在创建)
  • contains_key(“name”) 判断键是否存在
  • entry(key).or_insert(value) 检测一个键是否存在对应值，如果不存在，则为它插入一个值; 存在则不处理(或基于旧值更新 即 保留现有值,忽略更新)

检测一个键是否存在对应值，如果不存在，则为它插入一个值

基于旧值更新

  • 打印map中的键值对使用{:?}
  • for (key, value) in hashmap 循环

HashMap与所有权

对于像 i32 这样的实现了 Copy trait 的类型，其值可以拷贝进哈希 map。对于像 String 这样拥有所有权的值，其值将被移动而哈希 map 会成为这些值的所有者;

可以把对象的引用存入HashMap

match kv {
    Some((k,v)) => println!(“{}-{}”, k,v),
    None => println!(“无数据”),
}

.entry(k).or_insert(v) k键, v值 &mut V 如果key不存在插入值, 存在不进行操作，返回Map中value值的可变引用。 let v :&mut i32 = map.entry(1).or_insert(1);

keys() void Keys 获取map中所有key let keys = map.keys();
contains_key(&k) bool key的引用 判断key是否存在 true存在,false不存在 let ck = map.contains_key(&4);
clear void 清空map map.clear();
drain void Drain 清空map,并返回所有键值对的迭代器 let hhh = map.drain();
for v in hhh {
    println!(“{}-{}”, v.0, v.1);
}
extend(Iter) void Iter迭代器 向map中添加内容(相当于java中的putAll方法) map.extend(map2.iter());

clone_into(&mut map) void &mut map 清空入参中的map, 并把当前map中的key,value赋值给入参的map对象 map.clone_into(&mut map2);
