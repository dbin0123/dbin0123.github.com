集合(collections)

Rust 标准库中包含一系列被称为 集合（collections）的非常有用的数据结构。
Rust 的标准集合库提供了最常见的通用编程数据结构的高效实现。通过使用标准实现，两个库应该可以在不进行大量数据转换的情况下进行通信。

Rust 的集合可以分为四大类：

  • 序列(Sequences)：Vec, VecDeque,LinkedList
  • 地图(Map)：HashMap,BTreeMap
  • 列表(Sets)：HashSet,BTreeSet
  • 杂项(Misc)：BinaryHeap

Vector

vector 允许我们在一个单独的数据结构中储存多于一个的值，它在内存中彼此相邻地排列所有的值。vector 只能储存相同类型的值。

创建Vector

  • 使用 Vec::new 创建
  • 使用vec!宏创建
  • 使用Vec::from创建
  • 指定容量创建 Vec::with_capacity

插入数据

  • push方法
  • insert方法(指定下标插入)

读取 vector 的元素

  • 通过下标获取
  • 通过get方法获取
  • 通过pop移除获取

遍历 vector 中的元素

  • 使用for循环
  • 修改vec中的值
  • 变量并修改值(*号表示解引用)

使用枚举来储存多种类型

使用枚举来让vec实现类似于java中的List<Object>

附录Vector常用方法

创建vec

用法 示例 示例
Vec::new() 创建空vec let vec:Vec = Vec::new();
vec![元素] 创建vec let vec:Vec = vec![1,2,3];
Vec::from(iter) 通过迭代器创建vec let vec = Vec::from([1,2,3]);
String.into_bytes() 字符串转vec let v1: Vec = s.into_bytes();
[1,2,3].into() 数组转vec let v1: Vec = [1,2,3].into();
[1,2,3].into() 数组转vec let v1: Vec = [10;10].into_iter().collect();
Vec::with_capacity(capacity) 指定容量创建的vec let v1: Vec = Vec::with_capacity(10);

方法

方法 参数 返回值 描述 示例
push value void 向尾部添加元素 v.push(5);
vec[index] index(下标) value 读取 vector 的元素(index从0开始) ,超出范围会panick let v:i32 = vec[1];
get index(下标) Option<&Value> 读取 vector 的元素(index从0开始),超出范围不会panick let third: Option<&i32> = v.get(2);
get_mut index(下标) Option<&mut Value> 读取 vector 的元素(index从0开始) let third: Option<&mut i32> = v.get_mut(2);
len void usize 获取长度(数据) let len:usize = vec.len();
capacity void usize 获取Vec容量 let len:usize = vec.capacity();
pop void Option<&value> 删除最后一个元素并返回，如果为空则返回”无” let data:Option<&value> = vec.pop();

resize new_len, value void 指定长度并赋初始值 vec.resize(5, 0);
